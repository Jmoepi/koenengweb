
# Next.js Portfolio Website

This is a personal portfolio website built with Next.js, showcasing my projects and skills. The website is designed to be a modern, responsive, and performant platform to display my work and provide information about my professional background.

## Live Demo

[Link to live website](https://koenengweb.web.app)

## Features

*   **Responsive Design:** The website is fully responsive and works on all devices.
*   **Dynamic Project Showcase:** Projects are fetched from the GitHub API and displayed in a clean and organized manner.
*   **Live Demo and Code Links:** Each project card includes links to the live demo and the source code on GitHub.
*   **Contact Form:** A functional contact form for visitors to get in touch.
*   **Built with Next.js:** The website is built with the latest version of Next.js, ensuring optimal performance and a modern development experience.
*   **Styled with Tailwind CSS:** The user interface is styled with Tailwind CSS, a utility-first CSS framework.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Node.js (v20 or later)
*   pnpm (or npm/yarn)

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/Jmoepi/koenengweb.git
    ```
2.  Install NPM packages
    ```sh
    pnpm install
    ```
3.  Start the development server
    ```sh
    pnpm dev
    ```

## Scripts

*   `pnpm dev`: Starts the development server.
*   `pnpm build`: Builds the application for production.
*   `pnpm start`: Starts the production server.
*   `pnpm lint`: Lints the code.
*   `pnpm typecheck`: Runs the TypeScript compiler to check for type errors.
